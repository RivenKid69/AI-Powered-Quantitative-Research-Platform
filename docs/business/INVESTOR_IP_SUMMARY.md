### Patent #1: CVaR-RL Execution System

**Key Claims:**
1. Integration of Conditional Value-at-Risk into distributional RL value estimation
2. Twin Critics architecture with quantile outputs for risk-aware learning
3. Conformal prediction bounds on risk estimates
4. Sim-to-live parity monitoring system

**Prior Art Position:** No existing patents combine these elements. Academic work (Tamar 2015, Dabney 2018) treats risk as constraint, not embedded in value function.

### Patent #2: Unified Multi-Asset Simulation

**Key Claims:**
1. L1→L2→L3 fidelity scaling with consistent API
2. Auto-calibration from historical LOB data
3. Asset-agnostic execution provider architecture
